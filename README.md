# ascii-boxer
Change a picture into ascii art based on the 'lightness,' or 'value,' of each pixel using the various "shade" characters in Unicode.

## Getting Started

### Prerequisites
- Python 3.7
- PIL

### Running
`python3 ascii-boxer.py` and follow the prompts.
